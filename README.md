# 🚀 AI Shark Tank

AI Shark Tank is an AI-powered startup evaluation platform that analyzes startup ideas and provides funding predictions, valuation estimates, SWOT analysis, investor feedback, and business insights.

- Investor Panel Feedback
- Business Analytics Dashboard
